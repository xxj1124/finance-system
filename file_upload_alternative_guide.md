# 文件上传替代方法指南

## 问题描述

在当前阿里云轻量应用服务器控制台界面中，没有找到「文件」选项卡用于上传文件。

## 解决方案

以下是四种替代的文件上传方法，您可以选择其中一种最适合您的方式：

---

## 方法一：使用Workbench远程连接上传文件

### 步骤1：连接到Workbench

1. 在当前服务器概览页面顶部，点击「**远程连接**」按钮
2. 在弹出菜单中选择「**Workbench远程连接**」
3. 输入服务器登录密码，点击「**确定**」
4. 进入Workbench命令行界面

### 步骤2：使用Workbench上传文件

1. 在Workbench界面的顶部工具栏，找到并点击「**文件**」按钮
2. 在下拉菜单中选择「**上传文件**」
3. 在弹出的文件选择窗口中，导航到本地项目目录：`E:\trae项目\finance\`
4. 选择所有文件和文件夹（按Ctrl+A全选）
5. 点击「**打开**」按钮，开始上传
6. 等待上传完成

### 步骤3：移动文件到项目目录

文件上传后默认保存到 `/root` 目录，需要移动到项目目录：

```bash
# 创建项目目录（如果还没有创建）
mkdir -p /var/www/finance

# 移动文件
cp -r /root/* /var/www/finance/
```

---

## 方法二：使用命令助手上传文件

### 步骤1：进入命令助手

1. 在当前服务器概览页面顶部，点击「**命令助手**」选项卡
2. 进入命令助手页面

### 步骤2：上传文件

1. 在命令助手页面，找到「**文件管理**」区域
2. 点击「**上传文件**」按钮
3. 选择本地项目文件，点击「**打开**」
4. 等待上传完成
5. 文件会自动上传到服务器的 `/root` 目录

### 步骤3：移动文件到项目目录

```bash
cp -r /root/* /var/www/finance/
```

---

## 方法三：使用SCP命令从本地上传

### 步骤1：安装SCP工具（如果本地没有）

Windows 10/11系统已内置SCP命令，您可以直接使用PowerShell或命令提示符。

### 步骤2：执行SCP上传命令

1. 在本地计算机上打开PowerShell或命令提示符
2. 输入以下命令：
   ```bash
   scp -r "E:\trae项目\finance\*" root@8.138.228.54:/var/www/finance/
   ```

3. 按回车键执行
4. 输入服务器登录密码，按回车键
5. 等待文件上传完成

---

## 方法四：使用压缩包上传（推荐大文件）

### 步骤1：在本地压缩项目文件

1. 打开本地文件资源管理器
2. 导航到 `E:\trae项目\finance\` 目录
3. 选中整个 `finance` 文件夹
4. 右键点击，选择「发送到」→「压缩(zipped)文件夹」
5. 等待压缩完成，生成 `finance.zip` 文件

### 步骤2：上传压缩包

使用上述任意一种方法（Workbench、命令助手或SCP）上传 `finance.zip` 文件到服务器的 `/root` 目录。

### 步骤3：在服务器上解压文件

```bash
# 创建项目目录
mkdir -p /var/www/finance

# 进入项目目录
cd /var/www/finance

# 解压文件
unzip /root/finance.zip

# 如果解压后有嵌套目录，移动文件到正确位置
# 例如：如果解压后文件在 finance/finance/ 目录下，执行以下命令
# cp -r finance/* ./
# rm -rf finance/
```

## 验证文件上传成功

无论使用哪种方法，上传完成后都可以执行以下命令验证：

```bash
ls -la /var/www/finance/
```

如果能看到项目的所有文件和文件夹（如app、requirements.txt等），说明上传成功。
