# 财务管理系统

## 项目概述

这是一个面向小型企业的财务管理系统，提供完整的企业财务管理解决方案，包括：
- 企业初始化
- 采购管理
- 销售管理
- 费用管理
- 会计核算
- 财务报表
- 预算管理
- 税务处理

## 技术栈

- **编程语言**：Python
- **Web框架**：Flask
- **数据库**：SQLite
- **ORM**：SQLAlchemy
- **模板引擎**：Jinja2

## 项目结构

```
finance/
├── app/
│   ├── models/          # 数据库模型
│   ├── templates/       # 前端模板
│   ├── views/           # 业务逻辑
│   ├── __init__.py      # 应用初始化
│   └── config.py        # 配置文件
├── finance.db           # SQLite数据库文件
├── run.py               # 应用启动文件
├── start_server.bat     # Windows启动脚本
├── requirements.txt     # 依赖包列表
└── README.md            # 项目说明
```

## 安装和运行

### 环境要求

- Python 3.6 或更高版本
- pip 包管理器

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

#### 方法1：使用Windows启动脚本

双击运行 `start_server.bat` 文件，服务器将自动启动。

#### 方法2：命令行启动

```bash
python run.py
```

### 访问系统

服务器启动后，在浏览器中访问以下地址：

```
http://localhost:5000
```

## 系统使用

### 1. 企业初始化

首次使用系统时，需要完成企业初始化：
1. 访问系统首页，进入企业初始化页面
2. 填写企业基本信息：
   - 企业名称
   - 注册资金
   - 行业类型
   - 会计期间
   - 币种
3. 选择启用的模块
4. 点击"初始化企业"完成设置

### 2. 模块功能

#### 采购管理
- 供应商管理：添加、编辑、查看供应商信息
- 采购订单管理：创建、审批、跟踪采购订单

#### 销售管理
- 客户管理：添加、编辑、查看客户信息
- 销售订单管理：创建、审批、跟踪销售订单

#### 费用管理
- 费用报销：提交报销申请
- 费用审批：审批员工的报销申请

#### 会计核算
- 会计科目管理：维护会计科目体系
- 凭证录入：录入会计凭证

#### 财务报表
- 资产负债表：反映企业财务状况
- 利润表：反映企业经营成果
- 现金流量表：反映企业现金流动情况

#### 预算管理
- 预算编制：编制部门年度预算
- 预算执行：监控预算执行情况

#### 税务处理
- 税务申报：申报各种税种
- 税务管理：管理税务相关信息

## 系统配置

系统配置文件位于 `app/config.py`，可以根据需要修改：

- **数据库配置**：修改 `SQLALCHEMY_DATABASE_URI` 可以更换数据库
- **安全配置**：修改 `SECRET_KEY` 提高系统安全性
- **分页配置**：修改 `PER_PAGE` 设置每页显示记录数

## 数据备份

为了防止数据丢失，建议定期备份 `finance.db` 文件。

## 常见问题

### 问题1：服务器无法启动

**解决方法**：
- 确保已安装所有依赖包：`pip install -r requirements.txt`
- 检查端口5000是否被占用
- 查看命令行错误信息，根据提示解决问题

### 问题2：重启电脑后无法访问系统

**解决方法**：
- 运行 `start_server.bat` 重新启动服务器
- 检查服务器是否正在运行

### 问题3：系统功能异常

**解决方法**：
- 检查浏览器控制台是否有错误信息
- 查看服务器日志，了解具体错误
- 尝试清理浏览器缓存
